# Cypress Тесты для книжного приложения

## Описание проекта

Автоматизированные тесты для книжного приложения

## Установка

```bash
npm install

cypress/
├── e2e/
│   ├── auth.cy.js          # Тесты авторизации (3 теста)
│   ├── favorites.cy.js     # Тесты избранного (пропущены - нет книг)
│   └── first-test.cy.js    # Проверка загрузки страницы
├── support/
    ├── commands.js         # Кастомные команды
    └── e2e.js             # Настройки поддержки
Результаты тестов
Тест	Статус
auth.cy.js	✅ 3/3 пройдено
favorites.cy.js	⏭️ Пропущены (нет книг)
first-test.cy.js	✅ 1/1 пройдено
```
